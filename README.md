# 🧮 Simple Calculator

A clean and responsive calculator web app that performs basic arithmetic operations quickly and efficiently.

## 🚀 Live Demo

👉 https://calculator-simple-calculation.vercel.app/

## 📌 About the Project

This is a simple calculator built using HTML, CSS, and JavaScript. It allows users to perform basic calculations like addition, subtraction, multiplication, and division with a smooth and user-friendly interface.

## 💡 Features

* ➕ Addition
* ➖ Subtraction
* ✖️ Multiplication
* ➗ Division
* 🎯 Real-time calculation
* 📱 Responsive design

## 🛠️ Tech Stack

* HTML
* CSS
* JavaScript
